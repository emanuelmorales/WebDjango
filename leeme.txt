Panel de administracion:

Si vemos, anteriormente se creo el archivo db.sqilte3, esto quiere decir que vamos a trabajar con este tipo de BD, lo que no quita que trabajemos con otras DB. En este curso vamos a trabajar con sqlite3 que nos genera django.

si vemos el archivo settings.py, podremos ver que vienen instaladas app de forma predeterminada, entre ella se enceuntra admin

Si recordamos, en el url.py por defecto viene el path /admin que para ver a donde nos lleva esa ruta, levantamos el admihne ingresamos a http://127.0.0.1:8000/admin nos lleva al login de admin por que aún no hemos creado un superadmin para poder ingresa, para ello veremos a continuacion como crear ese usuario.

Debemos ejecutar el comando que crea las tabla ejecutando el siguiente comando
py manage.py migrate

ahora vamos a crear el superusuario con el comando
py manage.py createsuperuser
uss: (admin) por defecto nos coloca un nombre, pero podemos escribir el que queremos.
email: colocamos un email (admin@gmail.com)
password: ingresamos una contraseña (admin)
con esto ya hemos creado el superuser

Ahora si levantamos nuevamente el servidro ya vamos a http://127.0.0.1:8000/admin podemos ingresar con estas credenciales.

Una vez dentro, podremos ver el panel de administrador, y si ingresamos al partado users, podremos ver el usuario administrador que acabamos de crear. Desde aquí podremos administrar nuestros registros de la BD.

Más adelante, si quisieramos, podremos modificar el aspecto de este panel. 

Ahora si queremos podemos agregar un nuevo usuario o un nuevo grupo, modificar una contraseña o cualquier dato de un usuario ya creado, etc.
